Linter for CudaLint.
Supports CSS lexer.
Requires Node.js in PATH.

Uses "csslint" Node package, you must install it to the home folder:
~/node_modules/csslint/dist/cli.js

Use this terminal command from the home folder:
$ npm install csslint
